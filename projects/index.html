QuotesAPI Bot

QuotesAPI Bot - Бот, изначально созданный для себя чисто потому что хотелось иметь под рукой инструмент для генерации цитат, интегрированный с моим API для цитат.

Бот на данный момент насчитывает около 500 чатов, 11000 сгенерированных цитат и около 500 стикерпаков с цитатами пользователей.

rsAPI

rsAPI - Сборник всяких API, преимущественно для ботов Telegram, разрабатываемый rfoxxxyz studio.

На данный момент насчитывается около 12000 сгенерированных цитат и 1200 дистортнутых картинок.

rsPay

rsPay - Платежная система для получения оплаты и массовых выплат в проектах, разрабатываемых rfoxxxyz studio.

На данный момент имеет 3 подключенных проекта и находится в закрытом бета тесте.
